# CAMBIAR ÚNICAMENTE LA TIPOGRAFÍA DE LA PÁGINA WEB

Necesito cambiar solamente las tipografías de la página web para que tenga una apariencia más elegante, moderna y premium.

## INSTRUCCIÓN PRINCIPAL Y OBLIGATORIA

No rediseñes la página.

No cambies nada que no esté relacionado directamente con la tipografía.

No cambies:

* Colores
* Fondos
* Imágenes
* Videos
* Logotipo
* Secciones
* Orden de las secciones
* Contenido
* Textos
* Botones
* Enlaces
* Bordes
* Sombras
* Animaciones
* Márgenes
* Espaciados
* Tamaños de contenedores
* Posición de los elementos
* Diseño móvil
* Panel administrativo
* Funcionalidades
* Formularios
* Base de datos

El único cambio solicitado es aplicar las nuevas tipografías correctamente en toda la página.

---

# TIPOGRAFÍAS QUE DEBES UTILIZAR

## Para títulos

Utiliza:

**Cinzel**

Cinzel debe aplicarse solamente en:

* Título principal del Hero
* Títulos de secciones
* Títulos de servicios
* Títulos de paquetes
* Títulos de promociones
* Títulos de tarjetas
* Títulos de formularios
* Títulos de páginas legales
* Encabezados H1
* Encabezados H2
* Encabezados H3
* Frases importantes que actualmente funcionan como títulos

Pesos permitidos:

* 500
* 600
* 700

No utilices Cinzel en párrafos largos, formularios, menú ni botones.

---

## Para textos generales

Utiliza:

**Montserrat**

Montserrat debe aplicarse en:

* Menú de navegación
* Párrafos
* Descripciones
* Botones
* Formularios
* Campos de entrada
* Selectores
* Mensajes de confirmación
* Preguntas frecuentes
* Testimonios
* Información de contacto
* Footer
* Chat de inteligencia artificial
* Panel administrativo
* Textos legales
* Etiquetas
* Enlaces
* Textos pequeños

Pesos permitidos:

* 400 para párrafos
* 500 para menú y etiquetas
* 600 para botones y elementos importantes
* 700 solamente cuando sea necesario destacar información

# IMPORTACIÓN DE LAS FUENTES

Carga las fuentes correctamente desde Google Fonts o mediante el sistema de fuentes que ya utiliza el proyecto.

Puedes usar:

```html
<link
  rel="preconnect"
  href="https://fonts.googleapis.com"
/>

<link
  rel="preconnect"
  href="https://fonts.gstatic.com"
  crossorigin
/>

<link
  href="https://fonts.googleapis.com/css2?family=Cinzel:wght@500;600;700&family=Montserrat:wght@400;500;600;700&display=swap"
  rel="stylesheet"
/>
```

Si el proyecto utiliza React, Vite, Next.js, Tailwind o alguna librería de fuentes, integra las tipografías siguiendo la arquitectura existente.

No agregues la importación varias veces.

---

# VARIABLES DE TIPOGRAFÍA

Define variables globales para evitar estilos duplicados:

```css
:root {
  --font-heading: "Cinzel", Georgia, serif;
  --font-body: "Montserrat", Arial, sans-serif;
}
```

Aplica Montserrat como tipografía general:

```css
body {
  font-family: var(--font-body);
}
```

Aplica Cinzel solamente a títulos:

```css
h1,
h2,
h3,
.section-title,
.hero-title,
.service-title,
.package-title,
.promotion-title,
.card-title,
.form-title,
.legal-page-title {
  font-family: var(--font-heading);
}
```

Aplica Montserrat al resto de los elementos:

```css
p,
a,
button,
input,
textarea,
select,
label,
nav,
li,
span,
small,
blockquote {
  font-family: var(--font-body);
}
```

Adapta los selectores a las clases reales que ya existen en el proyecto.

No cambies los nombres de las clases actuales si no es necesario.

# TÍTULO PRINCIPAL DEL HERO

El título principal debe utilizar:

* Cinzel
* Peso 700
* La misma medida actual
* La misma posición
* El mismo color
* El mismo espaciado visual general

No cambies el tamaño actual del título salvo que la nueva fuente provoque un desbordamiento.

Si es necesario hacer un ajuste mínimo para evitar que el título se corte, utiliza `clamp()` sin cambiar visualmente el diseño.

Ejemplo:

```css
.hero-title {
  font-family: var(--font-heading);
  font-weight: 700;
  line-height: 1.1;
  letter-spacing: 0.02em;
}
```

No conviertas todos los títulos a mayúsculas automáticamente.

Respeta exactamente cómo están escritos actualmente.
