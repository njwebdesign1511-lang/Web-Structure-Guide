# AMPLIAR EL PANEL DE EDICIÓN EXISTENTE PARA CONTROLAR TODA LA PÁGINA WEB

Necesito modificar y ampliar el **panel de edición que ya existe actualmente en la página web**.

## INSTRUCCIÓN PRINCIPAL Y OBLIGATORIA

**No reemplaces el panel de administración existente.**

**No crees un panel nuevo.**

**No elimines ninguna función que ya existe.**

Debes revisar el panel actual, conservar su diseño, acceso, autenticación, rutas, base de datos y funciones existentes, y agregarle las herramientas necesarias para que desde ese mismo panel se pueda editar prácticamente toda la página web.

El objetivo es convertir el panel existente en un sistema completo de administración de contenido.

---

# 1. REVISIÓN ANTES DE MODIFICAR

Antes de hacer cambios:

1. Revisa todos los archivos relacionados con el panel actual.
2. Identifica cómo funciona el inicio de sesión.
3. Identifica las rutas protegidas.
4. Revisa cómo guarda actualmente la información.
5. Revisa las tablas o colecciones de la base de datos.
6. Identifica las funciones que ya existen.
7. Conserva todos los datos actuales.
8. Crea una copia de seguridad antes de modificar la estructura.
9. No cambies el acceso actual al panel.
10. No rompas ninguna función existente.

Extiende el panel actual utilizando la misma tecnología y arquitectura del proyecto.
