# FASE 4: ANIMACIONES Y EXPERIENCIA PREMIUM

No cambies las animaciones existentes.

Solo corrige las que:

* Se corten
* Se congelen
* Se repitan incorrectamente
* Aparezcan demasiado rápido
* Afecten la lectura
* Provoquen movimientos bruscos

Las animaciones deben ser:

* Elegantes
* Suaves
* Profesionales
* Rápidas pero no agresivas
* Consistentes
* Optimizadas

Utiliza preferiblemente:

* Fade-in
* Slide-up suave
* Elevación ligera de tarjetas
* Zoom mínimo en imágenes
* Transiciones de entre 250 y 600 milisegundos
* Animaciones activadas solamente cuando el elemento entra en pantalla

No utilices:

* Parpadeos
* Giros exagerados
* Movimientos infantiles
* Efectos que distraigan
* Animaciones constantes sin propósito
* Animaciones que afecten el rendimiento

Respeta la configuración:

```css
@media (prefers-reduced-motion: reduce)
```

Cuando el usuario prefiera menos movimiento, reduce o elimina las animaciones.
